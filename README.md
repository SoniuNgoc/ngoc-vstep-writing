# Ngọc VSTEP Writing Lab

Website luyện Writing VSTEP gồm Task 1 và Task 2, đồng hồ, đếm từ, lưu bài, nhận biết form, sửa bài Anh–Việt và giao diện sáng/tối.

## Mở trên máy

Chạy `START_WEB.bat` trên Windows, hoặc mở terminal tại thư mục này và chạy:

```bash
python -m http.server 8080
```

Sau đó mở `http://localhost:8080`.

## Đưa lên mạng

Xem hướng dẫn chi tiết trong `DEPLOY_ONLINE.md`.

## Sửa nội dung

Xem `EDIT_GUIDE.md`.

## Lưu ý

- Không đưa API key vào mã nguồn hoặc GitHub.
- Điểm và góp ý trên web chỉ phục vụ luyện tập, không phải điểm thi chính thức.
